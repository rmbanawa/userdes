Making First Web Application

REFLECTION # 3 by Rovyyc

In making my first web application in User Design class, for me, I did not find it difficult because I’ve already been making a web application when I was in Senior High in APC. They taught me basic JavaScript programming, database system and bootstrap, that is why I didn’t have any issue in making my web application. But still, I need to learn more about making a web application because technology evolved so fast, and it is important the we learn new technology.

This is my first time using Heroku and using a GitHub repository, it is useful for web developer to test their static HTML web pages and I can view my application through Heroku app. Still, I need to improve my design, add some navigation, proper color, typography and put some images. In order to make a good web application, it should be easy to understand and user-friendly because you’re not the only one who will access your website, in the future it can be your customers, or it depends on your audience.

Overall, it is fun, and I enjoy making my web application because it helps me to recall my knowledge about web development.

Userinyerface Quiz

REFLECTION # 4 by Rovyyc

I thought our quiz would be a typical quiz like other subjects, when I opened the userinyerface website, I don’t know how it works or the mechanics of the game. Then I understand that this quiz is like a puzzle that you need to solve to go to another level. The technic in this quiz is you need to read the instructions and use your logic to solve the problem. In my first attempt, I’ve finished the quiz in six minutes and nine seconds, which is not bad because some of my classmates finished it in 10 minutes above.

Since our class is all about user interface, I learned that we have to design our interface properly and make it simple so that the user will not have a problem in using your application, just like what happened to me, I got frustrated in userinyerface because I don’t what should I clicked and it keeps having an error. So, the moral lesson in this quiz is we should create a good user interface so that the users can properly use the application.

First Half of USERDES

REFLECTION # 6 by Rovyyc

In the first half of USERDES class was fine, although we have many activities, reflection every week, and projects to finish. As of now, I learned a lot, especially in making an interface before I just develop as long as the app run and working it’s fine with me without considering the design and interface. Now, I learned that the user interface has an important role in making a project not just in app because some people will use it and it should be user-friendly in order to use it with different kinds of people.

The only challenge that I’ve encountered so far is the amount of task we need to finish. Time management is very important in order to finish a different task because of it, I managed to pass all requirements. Also, Because I have a background in web development, I felt it was easy for me and that’s why I didn’t have any problem during the developing of personal website. I was chilling because I don’t want to stress myself in school activities, the more you feel comfortable in doing the task, the more you can finish it faster.

My expectation on finals, more lessons and technique on making a good user interface and upgrading personal website. Things that we can work on, the android studio making of layout, especially about the constraint and relative thing, I don’t know how it works.
